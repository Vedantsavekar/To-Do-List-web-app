# 📝 To-Do List App

A simple and modern To-Do List web application.

## 🚀 Features
- Add tasks
- Mark tasks as completed
- Delete tasks
- Data stored using localStorage
- Responsive design

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## 💡 How to Use
1. Enter your task
2. Click on Add button
3. Mark tasks as completed using checkbox
4. Delete tasks when done
